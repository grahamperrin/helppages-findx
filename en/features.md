## Quick searches:
You can do quick private searches on Findx by using the exits feature:

| !exit | Destination |
| ------------- | ------------- | 
| !w  | Web | 
| !s  | Shopping | 
| !i  | Images | 
| !m  | Maps | 
| !v  | Video |
